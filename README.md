# 무엇이 달라졌나요?
- 버프와 너프 자동 불러오기 개선

기존 버전에서는 자신의 버프와 너프를 찾는 곳이 분리되어 있었다면 이번 v3에는 이를 버프 및 너프 수치 입력 창과 연계하여 자동으로 수치가 입력되도록 하였습니다. 또한 버프와 너프 내역을 보여주며 어디에서 버프와 너프를 얻었는지 간편하게 알 수 있습니다.

- 업데이트 주기 개선

기존 버전에서의 버프와 너프 정보 업데이트는 간혹 신인 선수의 정보를 찾을 수 없기도 합니다. 이를 개선하여 v3에서는 스탯 계산과 동시에 정보 업데이트를 시행하여 신인 선수 분들도 자신의 정보를 쉽고 빠르게 찾아보셨으면 합니다.

- 노화 너프 누적 적용

기존 버전에서는 노화 너프가 누적 적용이 되지 않던 문제를 해결하였습니다. 나이 슬라이드 바를 이용해 노화 너프를 적용하세요!

# 사용법
선수명을 입력하여 버프와 너프 정보를 불러온 뒤, 밑의 방문, 게시글, 댓글 수를 입력하여 최종 계산된 결과를 확인하세요.

직접 나이와 버프, 너프를 입력하고 싶으시다면 입력 가능합니다. 상무 등으로 신인 버프가 잠시 중단되었다면 신인 버프 연장 체크박스를 통해 21살~24살까지로 버프 범위를 바꿀 수 있습니다.
